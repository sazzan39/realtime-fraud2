# 💳 Real-Time Fraud Detection System

An AI-powered system that detects fraudulent financial transactions in real-time using Machine Learning and streaming architecture.

-----

##  Overview

Most fraud detection projects focus only on model accuracy using static datasets.

This project goes a step further.

It simulates a **real-world fraud detection pipeline** where transactions are processed instantly, analyzed using a trained ML model, and visualized through a live dashboard.



## ⚡ Key Features

- Real-time transaction streaming using Kafka  
-  Machine Learning-based fraud prediction  
-  Live dashboard with alerts and analytics  
-  Instant classification (Fraud / Normal)  
-  Risk probability tracking  
-  Continuous transaction monitoring  



## 🏗️ System Architecture

| Component | Technology |
|----------|-----------|
| Streaming | Apache Kafka |
| Backend | Python |
| Machine Learning | Random Forest (scikit-learn) |
| Visualization | Streamlit |
| Data Processing | Pandas, NumPy |
